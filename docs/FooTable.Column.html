class  FooTable. Column  inherits FooTable.Class

new FooTable.Column(instance, cell, definition) → {FooTable.Column}

The column class containing all the properties for columns.
Parameters
Name Type Description
instance FooTable.Instance The parent FooTable.Instance this component belongs to.
cell HTMLElement The column header cell element.
definition object An object containing all the properties to set for the column.
Returns
FooTable.Column

Members

$headerCell :jQuery

The jQuery cell object for the column header.

direction :string = null

The direction to sort if the FooTable.Column#sorted property is set to true. Can be "ASC", "DESC" or NULL.

ellipsis :boolean = false

Whether or not to force a column to hide overflow with an ellipsis.

filterable :boolean = true

Whether or not the column can be used during filtering. Added by the FooTable.Filtering component.

formatter :function = null

A function used to format the columns value into a string to display. If NULL the plugin uses the cell values' default toString result. The column format function is passed the value obtained by the parser for the column and must return a string used to display the value in the cell, this result can be an HTML string.
Example

The below shows a formatter to convert a date or a ticks value to a string representation to display in a cell.

format: function(value){
	var months = ['Jan', 'Feb', 'Mar', 'Apr', 'May', 'Jun', 'Jul', 'Aug', 'Sep', 'Oct', 'Nov', 'Dec'];
	if (typeof value == "number") value = new Date(value);
	return value instanceof Date ? value.getDate() + ' ' + months[value.getMonth()] + ' ' + value.getFullYear() : null;
}

hidden :boolean = false

Whether or not this column is hidden from view and appears in the details row. This is set by the plugin.

hide :string = null

A comma separated string of breakpoint names that specify when the column will be hidden.

index :number = -1

The index of the column in the table. This is set by the plugin.

instance :FooTable.Instance

The FooTable.Instance for the column.

maxWidth :number = null

Specifies the maximum width for the column.

minWidth :number = null

Specifies the minimum width for the column.

name :string = null

The name of the column. This name must correspond to the property name of the JSON row data.

parser :function = jQuery.noop

The parse function for this column. This is set by the plugin.

pk :boolean = false

Whether or not the column is the primary key for the row.

sortable :boolean = true

Whether or not the column can be sorted.

sorted :boolean = false

Whether or not the column is sorted.

sorter :function = jQuery.noop

The sort function for this column. This is set by the plugin.

title :string = null

The title to display in the column header.

type :string = "text"

The type of data displayed by the column.

visible :boolean = false

Whether or not this column is completely hidden from view and will not appear in the details row.

width :number = null

Specifies the width for the column.

wrap :boolean = false

Whether or not to force a column to wrap overflow onto a new line. Overrides the FooTable.Column#ellipsis option.
